Proxy #

Proxy 作用 #

Proxy 是 ES6 新增的全局对象，用于修改某些操作的默认行为。可以理解为在目标对象之前做一层“拦截”，外部对该对象的访问，都必须经过这层拦截。

写一个简单的例子：

 let user = {
    name:'dingyi',
    age:18
 }
 let obj =  new Proxy(user,{
    get:function(target,key,receiver){
        return Reflect.get(target,key)+',代理生效了'
    },
    set:function(target,key,value,receiver){
        return Reflect.set(target,key,value,receiver)
    }
 })

obj.name // 'dingyi,代理生效了';
obj.name = 'dinger'
user.name // 'dinger'

上述对 user 对象架设了一层拦截，定义了 set 和 get 方法，根据打印发现默认的行为被覆盖了。

实例方法（只举例常用） #

（1）get(target, propKey, receiver) 拦截对象属性的读取，比如 proxy.foo 和 proxy['foo']。 最后一个参数 receiver 是一个对象，可选，参见下面 Reflect.get 的部分。

示例：

let person = {name:"张三"}
let proxy  = new Proxy(person,{
    get:function(target,field){
       if(Reflect.has(target,field)){
        return Reflect.get(target,field)
       }else{
        console.error(`${field}属性不存在`)
       }
    }
})
proxy.name //张三
proxy.age // age属性不存在

这个例子不做过多介绍。

再举个例子，封装一个方法，实现数组可以负数索引。

  let arr = [1,2,3];
  function createMyArr(arr){
    let handler = {
        get:function(target,index,receiver){
            index<0 ? target[target.length-Math.abs(index)]:target[index];
        }
    }
    return new Proxy(arr,handler)
  }
  const myArr = createMyArr(arr);
  myArr[-1] //3

最后一个例子，实现一个生成各种 dom 节点的通用函数 dom

const dom = new Proxy({},{
    get(target,property){
        return function(attrs,...children){
             const el = document.createElement(property);
             for(let prop of Object.keys(attrs)){
                el.setAttribute(prop,attrs[prop])
             }
             for(let child of children){
                if(typeof child ==='string'){
                    child = document.createTextNode(child)
                }
                el.appendChild(child)
             }
             return el;
        }
    }
})
const el = dom.div({},
  'Hello,my name is',
  dom.a({href:'https://www.baidu.com'},'Dingyi'),
  '. I like:',
  dom.ul({},
    dom.li({},'js'),
    dom.li({},'css'),
    dom.li({},'html')
  )
);
document.body.appendChild(el);

（2）set(target, propKey, value, receiver)拦截对象属性的设置，比如 proxy.foo = v 或 proxy['foo'] = v，返回一个布尔值。

示例，一个简单的校验属性添加例子：

let validator = {
    set:function(obj,prop,value){
        if(prop === 'age'){
            if(typeof value!==number){
                throw new TypeError("age必须为整数")
            }
            if(value>200){
                throw new RangeError("没人能活到200岁")
            }
        }
        Reflect.set(obj,prop,value)
    }
}

let person = new Proxy({},validator)
person.age = '12';//error
person.age = 2222;//error
person.age = 200; // success

（3）has(target, propKey)拦截 hasProperty,而不是hasOwnProperty。

举个例子，隐藏内部私有属性不被发现

const obj = {
    name:'dingyi',
    _age:18
}

const proxy = new Proxy(obj,{
    has:function(target,field){
        if(field.startsWith('_')){
            return false
        }
        return Reflect.has(target,field)
    }
})

_age in proxy // false

for(let a in proxy){
    console.log(a)   //name  _ age
}

显然，has 对 in 生效，但是has 拦截对 for...in 循环不生效

（4）deleteProperty(target, propKey)拦截 delete proxy[propKey]的操作，返回一个布尔值。

举个例子：

  const person ={name:'dingyi',_age:18}
  const handler = {
    deleteProperty(target,field){
        if(field.startsWith('_')){
            throw new Error('不能删除私有属性')
        }
        Reflect.deleteProperty(target,field)
    }
  }

const proxy = new Proxy(person,handler)
delete proxy.name // success
delete proxy._age // error
delete proxy.age1 // false

（5）ownKeys(target) 拦截 Object.getOwnPropertyNames(proxy)、Object.getOwnPropertySymbols(proxy)、Object.keys(proxy)，返回一个数组。该方法返回对象所有自身的属性，而 Object.keys()仅返回对象可遍历的属性。

（6）getOwnPropertyDescriptor(target, propKey)拦截 Object.getOwnPropertyDescriptor(proxy, propKey)，返回属性的描述对象。

（7）defineProperty(target, propKey, propDesc)拦截 Object.defineProperty(proxy, propKey, propDesc）、Object.defineProperties(proxy, propDescs)，返回一个布尔值。

（8）preventExtensions(target)拦截 Object.preventExtensions(proxy)，返回一个布尔值。

（9）getPrototypeOf(target)拦截 Object.getPrototypeOf(proxy)，返回一个对象。

（10）isExtensible(target)拦截 Object.isExtensible(proxy)，返回一个布尔值。

（11）setPrototypeOf(target, proto)拦截 Object.setPrototypeOf(proxy, proto)，返回一个布尔值。如果目标对象是函数，那么还有两种额外操作可以拦截。

（12）apply(target, object, args)拦截 Proxy 实例作为函数调用的操作，比如 proxy(...args)、proxy.call(object, ...args)、proxy.apply(...)。

apply 方法会拦截函数的调用，call 和 apply 操作 举个例子：

   let handler = {
    apply:function(){
        return 'i am apply'
    }
   }
   function test(){
    return 'i am test'
   }

   const fn = new Proxy(test,handler)

   fn(); //'i am apply' 被handler中apply拦截

另一个例子：

let handler = {
    apply(target,ctx,args){
    console.log('args',args)
    console.log('arguments',...arguments)
      return Reflect.apply(...arguments) * 2
    }
}
function test(left,right){
    return left+right
}

const proxy = new Proxy(test,handler);

proxy(1,2)   //6
proxy.call(null, 3, 4) // 14
proxy.apply(null, [5,6]) // 22

每当执行 proxy 函数（直接调用或 call 和 apply 调用），就会被 apply 方法拦截。

（13）construct(target, args)拦截 Proxy 实例作为构造函数调用的操作，比如 new proxy(...args)。

实现一个观察者模式 #

const person = observable({
  name: '张三',
  age: 20
});
function print() {
  console.log(`${person.name}, ${person.age}`)
}
observe(print);
person.name = '李四';
// 输出
// 李四, 20

主要就是实现 observable 和 observe; 实现如下：

const queuedObservers = new Set();
const observe = (fn)=>queuedObservers.add(fn);
const observable= (obj)=>new Proxy(obj,{set})

function set(target,key,value,receivers){
    const result = Reflect.set(target,key,value,receivers);
    queuedObservers.forEach(observer=>observer());
    return result;
}

把观察者函数全部放进一个队列集合中，observable 拦截观察对象属性变化并通知观察者执行。
